1.PATIENT LOGIN

A registered patient can login to the application by using registered emailid and password,it will provide following functions.

1.1 Pharmacy selection

After login to the application the Patient have the option to select various pharmacy that are listed in the available pharmacy

1.2 Appointments

In this part a patient can view and create/modify the appointment with the pharmacist.while creating the appointment the patient can select date and slot after selecting both the patient want to confirm the slot,once this done patient can view the appointment and status

1.3 Settings

By clicking on seeting, patients can view their details like name, address etc. And patients are also provided with a option to update their email address. Patients also view the selected pharmacy details in the seeting page

1.4 Payment

Patients are provided with various payment options like "pay using cards,pay using other methods". Payments details are listed in payment history page and also payment status is also displayed in the payment page likr"pending,success"

1.5 Test audio/video

This web application provide the users to test their audio and video before starting the meeting,once the testing is done patient must be wait in the lobby for pharmacist to start the meeting.patients can select appointment their own wish

1.6 About the site

this option provides the basic details about the application.The user can view the 'terms and condition,privacy policy,commercial terms', by using "help" patients can register the query. While writing the query the user wants to provide some information like name,phone number ,address etc.The frequently asked questions by the users are also shown in this page

1.7 Okusuritechou plus

This option provided to access the NC Digital Medidation notebook application.If this application already installed then by clicking on the link user can navigate to NC digital meditation notebook application, if not installed it will navigate to installation page

1.8 Events

this option is provided to view the events created by the admin. If any events are there it will be displayed to the user in this section

1.9 Logout

Logged user can logout from their profile by using logout option.If a person click on logout it will ask for confirmation
