2 Admin login

registered admin can login to the application by using their user id and password,it will provide following functions

2.1 Pharmacy selection

Admin can search for a specific pharmacy by using pharmacy id, if required pharmacy found then admin can select it

2.2 Pharmacy operation

After selecting pharmacy admin can perfron following tasks

2.2.1 Appointment

admin can view and create/modify the appointment .while creating the appointment the admin can select patient id ,date and slot after selecting both the admin want to confirm the slot,once this done patient can view the appointment and status

2.2.2 Patient setting

Admin can search for a patient by using patient id, after selecting patient admin can able to update the patient details

2.2.3 Notification settings

This application also provide admin to set notifications when a new patient registration completed and also when a patient waiting in the lobby for a meeting with pharmacist

2.3 Test audio/video

with this web application admin can able to test their audio and video before starting the meeting.

2.4 FAQ

FAQ section helps the admin to find the frequently asked question by the patient, also this application provides an option to filter the patient queries based on their question

2.5 Events

this application provide a feature to admin to create events, Admin can create events with informations like start time ,end time,description and required details. Once admin created the event it will be visible for both patient and pharmacist

2.6 Logout

Admin can logout buy clicking on logout bar
